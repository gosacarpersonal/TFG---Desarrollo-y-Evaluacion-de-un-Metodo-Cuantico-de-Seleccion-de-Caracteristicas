# src/viz_core Directory Summary

## Purpose
Provides the Mandatory Editorial Benchmark Warmth Layer to style Matplotlib figures in a clean, warm, academic style.

## File Map
* [__init__.py](file:///home/gosacar/TFG%20-%20QFS%20Last%20Chance/src/viz_core/__init__.py): Package initialization.
* [config.py](file:///home/gosacar/TFG%20-%20QFS%20Last%20Chance/src/viz_core/config.py): Configuration options for the visualization system.
* [editorial_warmth.py](file:///home/gosacar/TFG%20-%20QFS%20Last%20Chance/src/viz_core/editorial_warmth.py): Main class and methods (e.g. `EditorialPalette`, `set_editorial_rcparams`, `apply_editorial_axes`) for the warm light editorial palette and layout styling.
* [insights.py](file:///home/gosacar/TFG%20-%20QFS%20Last%20Chance/src/viz_core/insights.py): Functions to extract narrative insights and display callouts.
* [layout.py](file:///home/gosacar/TFG%20-%20QFS%20Last%20Chance/src/viz_core/layout.py): Helper functions to arrange subplots and spacing.
* [theme.py](file:///home/gosacar/TFG%20-%20QFS%20Last%20Chance/src/viz_core/theme.py): Theme selectors and style overrides.

## Key Dependencies
* matplotlib
